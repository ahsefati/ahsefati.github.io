Multi-Layer NN

Afetr perceptron, it is clear that we can't estimate all the functions by this (i.e. XOR function can't be estimated by a simple perceptron). So, we need a nueral network with multiple layers which each layer and each node has its own weight. In this work, I implemented a Multi-layer Nueral Network from scratch and without using any python libraries. I tested the model using "California Housing Prices" dataset provided by Kaggle.
